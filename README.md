# WorkuaTesting
# This is the simple test automation project.
<p>Solution stack: Selenide, TestNG, Maven</p>
<p>Report Framework: Allure </p>

## To run tests and generate Allure report:

* run `mvn clean test`
* run `mvn site`

## To see a report:

1. run `mvn jetty:run`
2. open `http://localhost:8080` in your browser
